# AI Travel Itinerary Planner

A full-stack web application that generates personalized day-by-day travel itineraries using the OpenAI API.

## Tech Stack
- Java, Spring Boot, Thymeleaf, OpenAI API

## Features
- Enter a destination, number of days, and your interests
- Generates a detailed day-by-day itinerary using AI
- Clean web interface built with Thymeleaf templates

## How to Run
1. Clone the repository
2. Add your OpenAI API key to `src/main/resources/application.properties`
3. Run with Maven: `./mvnw spring-boot:run`
4. Open your browser at `http://localhost:8080`

## Note
This project was built as part of a Java course. API key is not included for security reasons.
